# 2.1.3 Diagrama de Implantação

## Introdução

Os diagramas de implementação da UML desempenham um papel fundamental na representação da arquitetura física de sistemas, especialmente em aplicações distribuídas como o projeto EuAmoPiri. Por meio desses diagramas, é possível visualizar como os componentes de software e hardware se relacionam, além de compreender como o processamento está distribuído entre os diferentes elementos do sistema.

## Metodologia

Para a construção deste diagrama, utilizamos a ferramenta **Draw.io** e seguimos as diretrizes da Linguagem de Modelagem Unificada (UML).  
Nossa abordagem focou em mapear a arquitetura planejada, identificando os principais nós e artefatos de software, bem como as relações e protocolos de comunicação entre eles.
